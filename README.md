# Just a basic express app

## Features

* Webpack Build Ready
* Babel & ES2015
* Mongoose
* Mocha for tests
* ESlint
 
## Install
```
npm install
```

## Scripts

#### Start the project for development
```
npm run dev
```

#### Run All tests
```
npm test
```

#### Run ESlint
```
npm run lint
```

#### Remove dist and tmp
```
npm run clean
```

#### Build the API for production
```
npm run webpack
```
